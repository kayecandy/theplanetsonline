The Planets Online - Teacher's Guide

While this web site has been designed to allow the viewer to mix and match different commentaries and explore on their own, in a teaching context it’s helpful if everyone is seeing the same thing. This can be accomplished by specifying options at the end of the web address.

For example, to send out a link to a class that shows the Music Composition commentary, you add “/?commentary=musiccomp” making the following address: theplanetsonline.com/?commentary=musiccomp.

It’s possible to specify more than one commentary keyword, so if you wish students to see Music Composition and Music Performance, you would use a “+” sign, to make theplanetsonline.com/?commentary=musiccomp+musicperf. For a full list of commentaries see the table at the end of this page.

Additionally, you can draw attention to particular sections by specifying a start time. The following link starts playing ten minutes in: theplanetsonline.com/?start=00:10:00.

And if you wished to hide the orchestra leaving just the visuals, you can do that too. The following enables the ‘Images only’ video: theplanetsonline.com/?video=1. To enable the orchestra view use ‘video=0’.

These various options can be combined if you separate them with an ‘&’ sign. Here’s an example: theplanetsonline.com/?commentary=trivia&video=1&start=00:10:00

Would you like to write a commentary customized for your class? That’s easy. Just contact us for more details.
